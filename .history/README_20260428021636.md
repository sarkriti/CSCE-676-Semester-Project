# The Disease Gap: Analyzing Gender Disparity Patterns in California's Valley Fever Through Anomaly Detection 

Infectious disease outbreaks have the potential to create massive human and economic loss therefore there is an importance to take prebentive measures and pre-emptive action to manage devestating outbreaks. Additionally, not all diseases are spread equally, with some demographics and geographic locations being at more risk for particular infections. As I found out through my analysis of the dataset "Infectious Diseases by Disease, County, Year, and Sex" from the California Health and Human Services Open Portal, Coccidioidomycosis (Valley Fever) is a disease that largely affects men by a dispoportionate amount and is primarily concentrated in counties within the San Joaquin region of the state. By applying anomaly detection methods such as Isolation Forest, Local Outlier Factor, and KNN, I set out to explore patterns illustrating the unusual gender disparity of the disease.

## Dataset
The dataset [Infectious Diseases by Disease, County, Year, and Sex](https://data.chhs.ca.gov/dataset/infectious-disease) from the California Health and Human Services Open Portal is the dataset used for this project. The dataset is a report of various diseases by county from 2001-2023. There is one entry for the male cases, one entry for the female cases, and an entry for "total" cases for each disease-county-year. 

The preprocessing steps are as following. To address missing values for the cases field, we removed all null values. We categorically encoded the "sex" field and applied normalization using StandardScalar. Feature engineering with the help of LLMs tools such as Google Gemini and Perplexity was a key part of the process. We created two fields that flagged instances where the case count was small or zero which would be dropped when performing anomaly detection. Using the male, female, case numbers, and population data, we feature engineered a male_rate, female_rate, male-female rate ratio data field to be used to analyze gender disparity patterns. Additionally to reduce extreme outlier influence before scaling our data, we log transformed our total_cases field, total_population field, and rate_ratio field. Additionally a variable representing the rate difference and the absolute value of the rate difference were created as well.
